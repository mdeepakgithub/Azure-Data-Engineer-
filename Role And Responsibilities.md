## Role and Responsibilities

- Designed and developed **end-to-end ETL pipelines** using **Azure Data Factory** for ingesting data from **SQL Server, REST APIs, and flat files into ADLS Gen2**.

- Built and optimized **data transformation workflows in Azure Databricks (PySpark)** using **Delta Lake** for structured and semi-structured data.

- Implemented **incremental loading (CDC)** and **schema evolution** to handle evolving datasets.

- Developed **data models in Azure Synapse** for analytics consumption layers and **Power BI dashboards**.

- Automated **pipeline deployment using CI/CD** in **Azure DevOps and GitHub Actions**.

- Conducted **performance optimization** using **Z-ordering, caching, partitioning, and cluster tuning**.
